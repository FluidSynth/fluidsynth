# Settings Reference

FluidSynth settings are organised into the following groups.
Each setting has a name in *group.name* dotted notation, a type,
a default value, and an optional range.

- [Audio driver settings](audio.md)
- [MIDI driver settings](midi.md)
- [MIDI player settings](player.md)
- [Shell (command line) settings](shell.md)
- [Synthesizer settings](synth.md)
