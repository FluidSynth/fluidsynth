\doxysubsubsection{Detailed Description}
@endlifecycle

TCP socket server for a command handler.

The socket server will open the TCP port set by settings\+\_\+shell\+\_\+port (default 9800) and starts a new thread and \doxylink{group__command__handler}{Command Handler} for each incoming connection.

\begin{DoxyNote}{Note}
The server is only available if libfluidsynth has been compiled with network support (enable-\/network). Without network support, all related functions will return FLUID\+\_\+\+FAILED or NULL. 
\end{DoxyNote}
