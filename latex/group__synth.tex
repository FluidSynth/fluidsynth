\doxysubsection{Detailed Description}
Sound\+Font synthesizer

You create a new synthesizer with \doxylink{group__synth_ga4a70cc91e3203ef216f7fcc195c5c9b0}{new\+\_\+fluid\+\_\+synth()} and you destroy it with delete\+\_\+fluid\+\_\+synth(). Use the fluid\+\_\+settings\+\_\+t structure to specify the synthesizer characteristics.

You have to load a Sound\+Font in order to hear any sound. For that you use the fluid\+\_\+synth\+\_\+sfload() function.

You can use the audio driver functions to open the audio device and create a background audio thread.

The API for sending MIDI events is probably what you expect\+: fluid\+\_\+synth\+\_\+noteon(), fluid\+\_\+synth\+\_\+noteoff(), ...
