\doxysubsection{Detailed Description}
Functions for managing audio drivers and file renderers.

The file renderer is used for fast rendering of MIDI files to audio files. The audio drivers are a high-\/level interface to connect the synthesizer with external audio sinks or to render realtime audio to files. \input{group__audio__driver}
