\doxysubsubsection{Detailed Description}
Functions for managing MIDI drivers.

The available MIDI drivers depend on your platform. See settings\+\_\+midi for all available configuration options.

To create a MIDI driver, you need to specify a source for the MIDI events to be forwarded to via the \doxylink{group__Types_ga61c72b76e3ee344637994c3071f74d94}{fluid\+\_\+midi\+\_\+event\+\_\+t} callback. Normally this will be either a \doxylink{group__midi__router}{MIDI Router} via fluid\+\_\+midi\+\_\+router\+\_\+handle\+\_\+midi\+\_\+event() or the synthesizer via fluid\+\_\+synth\+\_\+handle\+\_\+midi\+\_\+event().

But you can also write your own handler function that preprocesses the events and forwards them on to the router or synthesizer instead.
