\doxysubsubsection{Detailed Description}
Interactive shell to control and configure a synthesizer instance.

If you need a platform independent way to get the standard input and output streams, use fluid\+\_\+get\+\_\+stdin() and fluid\+\_\+get\+\_\+stdout().

For a full list of available commands, type {\ttfamily help} in the shell.
